
Javascript 중단점을 활용한 디버깅

: 여기서 중단점(Breakpoint)은
  스크립트의 동작을 잠시 멈추도록하여 디버깅이 가능하게 해줌


< 단축키 >

F8 : 중단점을 그만하고 코드 계속 실행
F10 :  다음 함수 코드로 이동
F11 : 함수의 내부 코드로 이동
F11 + Shift : 현재 함수에서 빠져나와 다음 함수로 이동


+) F10 다음 실행 코드의 이동(Step over)

F11 키를 사용하면 해당 함수의 내부로 이동(Step into)
계속해서 디버깅이 필요한 경우 사용

만약 모두 확인하여 디버깅을 종료할 경우라면
F8 을 누르고 중단 또는 다음 중단점으로 이동



1. 작성한 코드 중 필요한 부분에 debugger 키워드 추가
  : debugger;

  중단된 상태에서 마우스를 testVal 위치에 오버하면
  어떤 값이 할당되었는지 툴팁에 나타나 확인 가능


2. 브라우저의 개발자도구의 소스 코드에 설정 (개발자 도구 및 콘솔창 이용)

 : 숫자를 클릭하여 설정
 
   이때 직접 디버깅할 코드를 찾아
   해당하는 라인의 좌측 영역을 클릭하면 중단점을 설정 할 수도 있음
   = 코드에 debugger; 넣은 것과 동일하게 동작
  
+) 동시에 여러 개 설정 가능
