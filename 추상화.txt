
추상화

할당 연산자
: =

복합 할당 연산자
: +=, -=, *=

증가, 감소 연산자
: ++, --
  (++ 의 위치에 따라 먼저 증가하고,
   나중에 증가하고의 차이 유의)


  // 다음 세 줄은 같은 의미
  x = x + 1;
  x += 1;
  x++;

  // 다음 세 줄은 같은 의미
  x = x - 1;
  x -= 1;
  x--;


return문
: return ;

- 어떤 값을 되돌려주는 output의 역할
- 함수의 실행을 중단하는 역할
  return 후, 작성된 코드는 Dead Code
