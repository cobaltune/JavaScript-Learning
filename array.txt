
배열 (Array)
: type of 연산자로 배열의 자료형을 출력해보면 ‘object’ 출력
⇒ 배열도 javascript에서 미리 만들어둔 조금 특별한 객체의 한 종류

// Javascript의 거의 모든 것이 객체 = 배열도 객체

- 랭킹처럼 순서가 있는 값을 만들 때
  (객체보다 배열을 활용하는 것이 좀 더 간결) 

- 순위가 아니더라도 순서가 있는 여러 값들의 묶음
  (순서에 신경 쓰지 말고, 단순하게 여러 값의 묶음이 필요한 경우에도 활용)


ex )
// 서울 공항철도 노선
let airportLines = [
  '인천공항 2터미널', '인천공항 1터미널', '검양', ...];

// 해리포터 시리즈
let harrypoterSeries = [
  '마법사의 돌','비밀의 방', '아즈카반의 죄수', ...];


// 코드잇 인기 코스 순위 (1등 ~ 4등) 나열한 객체

let courseRanking = {
	'1st' : '자바스크립트 프로그래밍 기초',
	'2nd' : 'Git으로 배우는 버전 관리',
	'3rd' : '컴퓨터 개론',
	'4th' : '파이썬 프로그래밍 기초'
};

위 객체를 배열로

let courseRanking = [
	'자바스크립트 프로그래밍 기초',
	'Git으로 배우는 버전 관리',

	'파이썬 프로그래밍 기초'

/* let abc = [
	‘element 1’,

- 배열은 중괄호 대신 대괄호
- 쉼표로 구분해서 순서대로 값을 입력

배열의 독특한 점은
대괄호 안의 각 요소별로 순서를 알려주는 숫자가 매겨짐

// index == propertyName
이 숫자값을 인덱스 라고 부름

배열의 요소를 가져오는 방법
// indexing (0 ~ …)
console.log (배열 이름 [index])  // 객체의 대괄효 표기법

** index는 1부터 시작하는 게 아니라 0부터 시작

ex )
console.log (courseRanking[0]);
